# Part 2: 함수형 프로그래밍

## 📚 학습 목표

- 순수 함수와 참조 투명성 이해
- 함수 합성을 통한 코드 재사용성 향상
- Option, Try, Either를 활용한 안전한 에러 처리
- Implicit을 활용한 타입 클래스 패턴 습득
- 고급 타입 시스템 (제네릭, Variance) 이해
